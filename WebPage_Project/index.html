What is a Virus?

According to Malwarebytes, A computer virus is a type of malware that attaches to another program which can replicate and spread after a person first runs it on their system. Viruses can be obtained multiple ways such as by email, by clicking a incorrect link or by downloading non trustworthy documents or PDF. Viruses are harmful for the system, they will slow down the performance, destroy data, and also log keystrokes.

Different Type of Viruses

1. File-Infecting Virus

Its a virus which infects a executable file and attaches itself to it. This type of virus is also reffered to as a parasitic virus it typically infects files with .exe or .com extensions

2. Macro Virus

This type of viruses are more commonly found in programs such as Microsoft Word and Excel. These viruses are usually stored in parts of the document and can spread to other files are sent to other computers, often through email.

3. Browser Hijacker

This virus targets and alters your browsers settings.It is often called a browser redirect virus because it sends your browser to malicious webpages you never intended to visit.

4. Web Scripting Virus

This type of virus is a sneaky one that targets popular websites.What it does is overwrite the code in a website and insert links that can install malicious software into the victims device. Web Scripting viruses can steal the victims cookies and use that information to make post on behalf of the victim in the infected site.

5. Resident Virus

This virus stores itself in the victims computer memory which allows it to infect files in the computer. This virus can interfere with your operating system leading to file and program corruption.
